# 更新日志

## v2.1.0 (2025-12-23)

### 🎨 命令行显示优化

#### 改进内容
- ✅ **代码块背景优化** - 采用深色渐变背景 (#1e1e2e → #181825)，提高可读性
- ✅ **字体优化** - 使用专业的等宽字体 (SF Mono, Monaco, Cascadia Code)
- ✅ **字号增大** - 从 0.875rem 提升到 1rem，更易阅读
- ✅ **行高增加** - 从 1.8 提升到 1.9，增加行间距
- ✅ **语法高亮** - 添加命令、参数、字符串的彩色高亮
- ✅ **命令提示符优化** - 使用绿色 (#a6e3a1) 并添加发光效果
- ✅ **边框和阴影** - 添加精细的边框和多层阴影，增强立体感

#### 颜色方案 (Catppuccin Mocha)
```css
背景: #1e1e2e → #181825 (深色渐变)
文本: #cdd6f4 (浅灰白色)
命令: #89b4fa (蓝色)
参数: #f9e2af (黄色)
字符串: #a6e3a1 (绿色)
提示符: #a6e3a1 (绿色)
```

#### 语法高亮
- `.command` - 命令名称 (蓝色 + 发光)
- `.param` - 参数 (黄色)
- `.string` - 字符串 (绿色)
- `.comment` - 注释 (灰色 + 斜体)
- `.symbol` - 特殊符号 (橙色)

#### 暗色模式优化
- 代码块背景更深 (#11111b)
- 边框更明显 (rgba(148, 163, 184, 0.3))
- 阴影更强 (rgba(0, 0, 0, 0.5))

#### 表格优化
- 表格标题使用深色渐变背景
- 表格代码使用相同的深色背景
- 悬停效果更明显
- 表格文本颜色优化

#### 滚动条样式
- 自定义滚动条样式
- 滚动条轨道半透明
- 滚动条滑块蓝色渐变
- 悬停时滑块更亮

### 📝 HTML 更新
所有代码块都添加了语法高亮标记：
```html
<div class="code-block">
    <span class="code-prompt">$</span> 
    <span class="command">npm</span> 
    <span class="param">install</span> 
    <span class="param">-g</span> 
    <span class="string">@letta-ai/letta-code</span>
</div>
```
